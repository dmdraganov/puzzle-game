Интерактивное приложение-пазл для мобильных устройств.

Требования:

1. Игровое поле с разбросанными элементами пазла
2. Возможность перемещать элементы касанием
3. Автоматическое притягивание к правильной позиции
4. Анимация при соединении элементов
5. Поддержка мультитач (перемещение нескольких элементов одновременно)
6. Система подсчета очков и времени
7. Эффекты при завершении пазла
8. Сохранение прогресса
9. Адаптация под разные размеры экрана
10. Тактильная отдача (вибрация) при правильном размещении
    Дополнительно:
    • Реализуйте разные уровни сложности
    • Добавьте звуковые эффекты
    • Создайте систему подсказок
    • Реализуйте режим "соревнование с временем"
